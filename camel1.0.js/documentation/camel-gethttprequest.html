Camel.GetHttpRequest

Camel.GetHttpRequest .

Members

_suc

Hold the success callback.

How to use:

var request = new Camel.GetHttpRequest();
request.addEventListener('ready', function() {
    alert("OK");
}, false);
console.log(request._suc);

_err

Hold the error callback.

How to use:

var request = new Camel.GetHttpRequest();
request.addEventListener('error', function() {
    alert("false");
}, false);
console.log(request._err);

_xhr

Hold the XMLHttpRequest object.

How to use:

var request = new Camel.GetHttpRequest();
request.Url('http://example.com/asset.json');
console.log(request._xhr);

_m

Hold the request type. Default, it is given "GET"

How to use:

var request = new Camel.GetHttpRequest();
console.log(request._m);

Methods

addEventListener(e, callback, t)

This method add the event callback. It has two events: ready and error

This method has three params:

  • e:string - The id string presents event type.
  • callback:function - The function executed when event orcurs.
  • t:boolean

How to use:

var request = new Camel.GetHttpRequest();
request.addEventListener('error', function() {
    alert("false");
}, false);
console.log(request._err);
