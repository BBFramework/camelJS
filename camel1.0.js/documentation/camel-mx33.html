Camel.Mx33

Camel.Mx33 is the 3x3 matrix math.

Members

mat

Hold the Float32Array(9).

How to use:

var vec1 = new Camel.Mx33(1.0, 2.0, 5.0, 0.0, 5.0, 0.0, 5.0, 0.0, 5.0);
console.log(vec1.mat);

Constructor

Camel.Mx33(x1, y1, z1, x2, y2, z2, x3, y3, z3)

The constructor of Camel.Mx33 has nice params:

  • x1:number - x1 value of the matrix.
  • y1:number - y1 value of the matrix.
  • z1:number - z1 value of the matrix.
  • x2:number - x2 value of the matrix.
  • y2:number - y2 value of the matrix.
  • z2:number - z2 value of the matrix.
  • x3:number - z3 value of the matrix.
  • y3:number - y3 value of the matrix.
  • z3:number - z3 value of the matrix.

How to use:

var mat1 = new Camel.Mx33(1.1, 2.1, 4.1, 0.0, 4.1, 0.0, 4.1, 0.0, 4.1);
console.log(mat1);
