Camel.Camera

Camel.Camera is the 4x4 matrix math.

Members

mat

Hold the Float32Array(16).

How to use:

var eye = new Camel.Vec3(0.0, 2.0, 0.0);
var look = new Camel.Vec3(0.0, 2.0, 6.0);
var up = new Camel.Vec3(0.0, 1.0, 0.0);
var cam = new Camel.Camera(eye, look, up);
console.log(cam.mat);

Constructor

Camel.Camera(eye, look, up)

The constructor of Camel.Camera has three params:

  • eye:Camel.Vec3 - The position of the Camera.
  • look:Camel.Vec3 - The target looking of the camera.
  • up:Camel.Vec3 - The up vector of the camera.

How to use:

var eye = new Camel.Vec3(0.0, 2.0, 0.0);
var look = new Camel.Vec3(0.0, 2.0, 6.0);
var up = new Camel.Vec3(0.0, 1.0, 0.0);
var cam = new Camel.Camera(eye, look, up);
console.log(cam);
