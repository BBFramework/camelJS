Camel.AssetManager

Camel.AssetManager loads and manages the assets.

Members

QueueType

Hold the array which manage the asset type.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.QueueType);

errorCount

Hold the number of error meeting.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.errorCount);

successCount

Hold the number of success meeting.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.successCount);

AssetList

Hold the array which manages asset completely loaded.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.AssetList);

Constructor

Camel.AssetManager(ImageExt, SoundExt, XhttpExt, VideoExt)

The constructor of Camel.AssetManager has four params:

  • ImageExt:string - The string presents for list of image type.
  • SoundExt:string - The string presents for list of sound type.
  • XhttpExt:string - The string presents for list of xhttp type.
  • VideoExt:string - The string presents for list of video type.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr);

parseExt(Path)

Receives the string present the extension of the asset.

The method has one param:

  • Path:string - The string presents for the path of asset.

Return string - The string present the file url.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.parseExt('Path/Asset.jpg'));

getType(Path)

Receives the number present the type of the asset.

The method has one param:

  • Path:string - The string presents for the path of asset.

Return number - The string present the file url.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.getType('Path/Asset.jpg'));

isQueued(Path)

Receives the boolean specify whether the asset had setuped.

The method has one param:

  • Path:string - The string presents for the path of asset.

Return Path - The boolean.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
console.log(assetMgr.isQueued('Path/Asset.jpg'));

QueueFile(Path)

Queues the file path into Asset Manager.

The method has one param:

  • Path:string - The string presents for the path of asset.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
assetMgr.QueueFile('Path/Asset.jpg');

QueueDownloadFile(Index, Type)

Download a file to Asset Manager.

The method has two params:

  • Index:number - Index of asset in the list.
  • Type:string - Type of asset.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
assetMgr.QueueFile('Path/Asset.jpg');
assetMgr.QueueDownloadFile(0, CAMEL_QUEUE_IMAGE);

QueueDownloadAll()

Download all asset.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
assetMgr.QueueFile('Path/Asset-01.jpg');
assetMgr.QueueFile('Path/Asset-02.jpg');
assetMgr.QueueDownloadAll();

getAsset(Path)

Get the asset from Asset Manager. If the asset is not exist, it is be downloaded to Asset Manager.

The method has one param:

  • Path:string - The string presents for the path of asset.

How to use:

var assetMgr = new Camel.AssetManager('jpg|png|gif');
assetMgr.getAsset('Path/Asset.jpg');
