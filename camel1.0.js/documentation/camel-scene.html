Camel.Scene

Camel.Scene is the 2x2 matrix math.

Members

mat

Hold the Float32Array(4).

How to use:

var vec1 = new Camel.Vec4(1.0, 2.0, 5.0, 0.0);
console.log(vec1.mat);

Constructor

Camel.Mx22(x, y, z, w)

The constructor of Camel.Mx22 has 4 params:

  • x:number - x-coord of the vector4.
  • y:number - y-coord of the vector4.
  • z:number - z-coord of the vector4.
  • w:number - w-coord of the vector4.

How to use:

var mat1 = new Camel.Mx22(1.1, 2.1, 4.1, 0.0);
console.log(mat1);

Methods

loadMXFloat()

Short description for this code

Params introduce:

  • aggr1:string - The id string of canvas developer want to use with Camel.
  • aggr2:object - The settings object developer want to config for Camel.

Return Float32Array(3) - the 3D vector value.

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
