Camel.Scene

Camel.Scene is the 3D scene.

Members

visible

Hold the flag specify whether the scene invisible or visible. Default, it is visible.

How to use:

var scene = new Camel.Scene();
console.log(scene.visible);

disable

Hold the flag specify whether the scene enable or disable. Default, it is enable.

How to use:

var scene = new Camel.Scene();
console.log(scene.disable);

particleHolder

The array hold the particle items. Default, it has 32 slots.

How to use:

var scene = new Camel.Scene();
console.log(scene.particleHolder.length);

Constructor

Camel.Scene(startCB, updateCB, renderCB, beforeRenderCB)

The constructor has four params:

  • startCB:function - Initialize the scene function.
  • updateCB:function - Update the scene function.
  • renderCB:function - Render the scene function.
  • beforeRenderCB:function - Before render function.

How to use:

var scene = new Camel.Scene();
console.log(scene);

Methods

Enable()

Enable the scene.

How to use:

var scene = new Camel.Scene();
scene.Enable();

Disable()

Disable the scene.

How to use:

var scene = new Camel.Scene();
scene.Disable();

addRenderer(renderer)

Add renderer to renderer list.

The method has one param:

  • renderer:object - The renderer used to render the scene.

How to use:

var scene = new Camel.Scene();
scene.addRenderer(CAMEL_RENDERER_TEXTURE);

pass(renderer)

Rendering the scene.

The method has one param:

  • renderer:object - The renderer used to render the scene.

How to use:

var scene = new Camel.Scene();
scene.pass(CAMEL_RENDERER_TEXTURE);

addChild(particle)

Add particle to the scene.

The method has one param:

  • particle:object - The object has be rendered by the scene.

How to use:

var scene = new Camel.Scene();
scene.addChild(children);

addLight(light)

Add the light to the scene.

The method has one param:

  • light:object - The light object has be rendered by the scene.

How to use:

var scene = new Camel.Scene();
scene.addLight(sunLight);
