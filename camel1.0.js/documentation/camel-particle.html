Camel.Particle

Camel.Particle class collects methods and members about particle.

Extended by Camel.Transform, Camel.Geometry, and Camel.Material.

Members

You can see members of Camel.Material, Camel.Geometry, and Camel.Transform for more information.

visible

Hold the visible which specifies whether this particle is enable or disable.

How to use:

var particle = new Camel.Particle();
console.log(particle.visible);

disable

Hold the disable which specifies whether this particle is enable or disable.

How to use:

var particle = new Camel.Particle();
console.log(particle.disable);

freezed

Hold the freezed which specifies whether this particle is freezed or free.

How to use:

var particle = new Camel.Particle();
console.log(particle.freezed);

Constructor

Camel.Particle()

A particle instance created by Camel.Particle class.

How to use:
