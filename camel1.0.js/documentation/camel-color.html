Camel.Color

Camel.Color class collects the methods and the members about color and material.

Members

cull

Hold the boolean specify whether this particle has 2-face material. Default, it is given by true value.

How to use:

var color = new Camel.Color();
color.cull = false;
console.log(color.cull);

alpha

Hold the alpha of material. Default, it is given by 100 value.

How to use:

var color = new Camel.Color();
color.setAlpha(50);
console.log(color.alpha);

shininess

The shininess of material.

How to use:

var color = new Camel.Color();
color.setShininess(5.0);
console.log(color.shininess);

color

The color value of material.

How to use:

var color = new Camel.Color();
console.log(color.color); // Should read only. (recommedation)

Constructor

Camel.Color()

How to use:

var color = new Camel.Color();
console.log(color);

Methods

setAlpha(value)

Allow sets the alpha of material.

This method has one param:

  • value:number - The number presents for the alpha, it has being limited between 0.0 and 100.0.

How to use:

var color = new Camel.Color();
color.setAlpha(50);
console.log(color.alpha);

setDiffuse(red, green, blue)

Allow sets the diffuse color of material.

This method has three params:

  • red:number - The number presents the red channel, it has being limited between 0.0 and 255.0.
  • green:number - The number presents the green channel, it has being limited between 0.0 and 255.0.
  • blue:number - The number presents the blue channel, it has being limited between 0.0 and 255.0.

How to use:

var color = new Camel.Color();
color.setDiffuse(255.0, 255.0, 255.0);
console.log(color.alpha);
