Camel.Geometry

Camel.Geometry about geometry in camel.

Members

vertexBuffer

Hold the buffer of vertex.

How to use:

var geo1 = new Camel.Geometry();
geo1.createCell(10.0);
console.log(geo1.vertexBuffer);

indicesBuffer

Hold the buffer of indice.

How to use:

var geo1 = new Camel.Geometry();
geo1.createCell(10.0);
console.log(geo1.indicesBuffer);

numberOfIndices

Hold the number of indice.

How to use:

Methods

createCell(size)

This method creates the cell for quad rendering.

This method has one param:

  • size:number - The number presents for size of quad.

How to use:

var geo1 = new Camel.Geometry();
console.log(geo1.createCell(10.0));

setModel(model)

This method set model from JSON to buffers.

This method has one param:

  • model:object - The model data JSON.

How to use:

var geo1 = new Camel.Geometry();
console.log(geo1.setModel("http://json.com/object-model.json"));
