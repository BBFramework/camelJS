GLOBAL Variables

GLOBAL Variables are the javascript variables which developer can access in every where in code.

CAMEL_WEBGL

This is the text string which contains for 'experimental-webgl'

CAMEL_SHADER_VERT

This is the text string which contains for 'x-shader/x-vertex'. It used to specify the shader type. In this case standing for vertex shader.

CAMEL_SHADER_FRAGMENT

This is the text string which contains for 'x-shader/x-fragment'. It used to specify the shader type. In this case standing for fragment shader.

CAMEL_ATTRIB

This is type of shader varible, In this case standing for attribute variable.

CAMEL_UNIFORM

This is type of shader variale, In this case standing for uniform variable.

CAMEL_QUEUE_IMAGE

This is file type which asset manager queue to import into Camel. In this case, it is type of image.

Camel accepts the types: '.jpg|.png|.gif'

CAMEL_QUEUE_SOUND

This is file type which asset manager queue to import into Camel. In this case, it is type of audio.

Camel accepts the types: '.mp3|.wav'

CAMEL_QUEUE_XHTTP

This is file type which asset manager queue to import into Camel. In this case, it is type of html.

CAMEL_QUEUE_VIDEO

This is file type which asset manager queue to import into Camel. In this case, it is type of video.

Camel accepts the types: '.mp4|.webM'

CAMEL_CREATE_CELL

This is a build-in entity creating flag. In this case, it is cell creation.

CAMEL_CREATE_BOX

This is a build-in entity creating flag. In this case, it is box creation.

CAMEL_CREATE_SPHERE

This is a build-in entity creating flag. In this case, it is sphere creation.

CAMEL_CREATE_CONE

This is a build-in entity creating flag. In this case, it is cone creation.

CAMEL_TYPE_OBJECT

This is a build-in abstractive type assigning flag. In this case, it is Object type.

CAMEL_TYPE_STRING

This is a build-in abstractive type assigning flag. In this case, it is String type.

CAMEL_TYPE_FUNC

This is a build-in abstractive type assigning flag. In this case, it is Function type.

CAMEL_MATH_EPSILON

This is a Math constant about Epsilon. It has 0.000001 value.

CAMEL_IL_FOUR

This is an input-layout type flag. In this case, it has a layout with four inputs.

CAMEL_IL_THREE

This is an input-layout type flag. In this case, it has a layout with three inputs.

CAMEL_IL_TWO

This is an input-layout type flag. In this case, it has a layout with two inputs.

CAMEL_IL_ONE

This is an input-layout type flag. In this case, it has a layout with one inputs.

NULL

This is a value type flag which presents null value.

TRUE

This is a value type flag which presents true value.

FALSE

This is a value type flag which presents false value.

INFINITY

This is a value type flag which presents -999,999,999 value.

UNFINITY

This is a value type flag which presents 999,999,999 value.

EMPTY

This is a value type flag which presents '' value.

UNSET

This is a value type flag which presents undefined value.

GL_COLOR_BUFFER_BIT

This is a buffer index which presents number of color bits.

GL_DEPTH_BUFFER_BIT

This is a buffer index which presents number of depth bits.

CAMEL_RENDERER_TEXTURE

It contains a renderer with texture only.

CAMEL_RENDERER_WORLD

It contains a renderer with texture, color, material, and light. This one works on PC system.

CAMEL_RENDERER_STANDARD

It contains a renderer with texture, and color only.

CAMEL_RENDERER_CAREM

It contains a renderer with texture only. This one used to render 2D graphics.

CAMEL_LIGHT_DIRECT

This is a light type flag which presents for the direct light.

CAMEL_LIGHT_POINT

This is a light type flag which presents for the point light.

CAMEL_STRIKE_ATTRIB

This is a attribute value which presents for the strike attribute.

CAMEL_OFFSET_ATTRIB

This is a attribute value which presents for the offset attribute.
