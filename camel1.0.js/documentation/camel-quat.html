Camel.Quat

Camel.Quat is the quaternion math.

Members

vec

Hold the Float32Array(4).

How to use:

var q = new Camel.Quat(1.0, 2.0, 5.0, 0.0);
console.log(q.vec);

sqlerp

Performs a spherical linear interpolation with two control points.

How to use:

var q = new Camel.Quat(1.0, 2.0, 5.0, 0.0);
console.log(q.sqlerp);

Constructor

Camel.Quat(i, j, k, q)

The constructor of Camel.Quat has 4 params:

  • i:number - i value of the quaternion.
  • j:number - j value of the quaternion.
  • k:number - k value of the quaternion.
  • q:number - q value of the quaternion.

How to use:

var q = new Camel.Quat(1.1, 2.1, 4.1, 0.0);
console.log(q);
