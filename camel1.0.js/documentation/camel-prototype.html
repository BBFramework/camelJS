Camel Prototype

Camel Prototype is a collection of members and methods in the Camel class.

Members

id

Hold the canvas id string.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.id);

element

Hold the canvas DOM element object.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.element);

gl

Hold the canvas context object.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.gl);

context

Hold the canvas context object.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.context);

lastTime

Hold the last time Camel used.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.lastTime);

clearColor

The array holds colors camel use to clear the context. This array has three elements presents for three channel R, G, B.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
engine.setClearColor(255, 0, 0);
console.log(engine.clearColor);

renderHolder

The array holds renderers camel use to render out from the context. Defaultly, this array has eight elements.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine.renderHolder);

Construct

Camel(CANVASElementID, Settings, Extensions, numberHolder)

The construct of Camel has 4 params:

  • CANVASElementID:string - The id string of canvas developer want to use with Camel.
  • Settings:object - The settings object developer want to config for Camel.
  • Extensions:array - An array collects WebGL extensions.
  • numberHolder:number - Number of renderer. Minimum is 4.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
console.log(engine);

Resize the size of context fix to the size of the window browser.

engine.getGLSL(c);

buildScene(renderer, startCB, updateCB, renderCB, beforeRenderCB)

Building the scene.

This method has 5 params:

  • renderer:object - The renderer used to render the scene.
  • startCB:callback - The start steps the scene.
  • updateCB:callback - The update when the scene has being rendered.
  • renderCB:callback - The render steps.
  • beforeRenderCB:callback - The steps before render.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
var scene = engine.buildScene(
    CAMEL_RENDERER_TEXTURE,
    function()
    {
        // Some start code of the scene...
    }
);

orderRender(renderer)

Registry the renderer into camel render machine.

This method has 1 param:

  • renderer:object - The renderer need to registry.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
engine.orderRender(CAMEL_RENDERER_TEXTURE);

buildRender(shaders, constants, beforeCB, finishCB)

Building the renderer.

This method has 4 params:

  • shaders:object - The shaders used to render the scene.
  • constants:object - The object contains the members are constants buffer of shader.
  • beforeCB:callback - The action has be excuted when the scene has being rendered.
  • finishCB:callback - The action has be excuted when the scene renders completely.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
var scene = engine.buildRender(
    [vs, fs],
    {
        'positionIn' : CAMEL_ATTRIB
        'texCoordIn' : CAMEL_ATTRIB
    }
);

engine.createTexture(textureAsset);

createRTT(name, width, height, red, green, blue, alpha, render)

Create dynamic texture.

This method has 8 params:

  • name:string - the name of texture.
  • width:number - the width of texture.
  • height:number - the height of texture.
  • red:number - the red channel of swap color.
  • green:number - the green channel of swap color.
  • blue:number - the blue channel of swap color.
  • alpha:number - the alpha channel of swap color.
  • render:object - the renderer used to render the scene.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
var tex = engine.createRTT('texName', 256, 256, 255, 0, 0, 100, renderer);

openRTT(buffer)

Begin render to frame buffer.

This method has 1 param:

  • buffer:object - The frame buffer shown as dynamic texture.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
engine.openRTT(frameBuffer);

closeRTT()

Close render to frame buffer.

How to use:

var engine = new Camel('canvas_id', {antialias:false}, [], 4);
engine.openRTT(frameBuffer);

// Render something at here...

engine.closeRTT();
