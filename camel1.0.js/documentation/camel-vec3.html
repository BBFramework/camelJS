Camel.Vec3

Camel.Vec3 is the vector 3D math.

Members

vec

Hold the Float32Array(3).

How to use:

var vec1 = new Camel.Vec3(1.0, 2.0, 5.0);
console.log(vec1.vec);

Constructor

Camel.Vec3(x, y, z)

The constructor of Camel.Vec3 has 3 params:

  • x:number - x-coord of the 3D vector.
  • y:number - y-coord of the 3D vector.
  • z:number - z-coord of the 3D vector.

How to use:

var vec1 = new Camel.Vec3(1.1, 2.1, 4.1);
console.log(vec1);

hermite(a, b, c, d, t, out)

Performs a linear interpolation between two control points.

This method has six params:

  • a:Camel.Vec3 - The first operand.
  • b:Camel.Vec3 - The second operand.
  • c:Camel.Vec3 - The third operand.
  • d:Camel.Vec3 - The fourth operand.
  • t:number - The time interpolation.
  • out:boolean | Camel.Vec3 - The out flag, if true it returns the new result, otherwise it change value of current object or change value of out object.

Returns linear interpolation 3D vector.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
var vec2 = new Camel.Vec3(5.0, 1.0, 2.0);
var vec3 = new Camel.Vec3(3.0, 2.0, 1.0);
var vec4 = new Camel.Vec3(1.0, 3.0, 6.0);
var vec5 = new Camel.Vec3(2.0, 2.0, 3.0);
var hermiteVec = vec1.hermite(vec2, vec3, vec4, vec5, dt);
