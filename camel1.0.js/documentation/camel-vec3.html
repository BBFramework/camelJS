Camel.Vec3

Camel.Vec3 is the vector 3D math.

Members

vec

Hold the Float32Array(3).

How to use:

var vec1 = new Camel.Vec3(1.0, 2.0, 5.0);
console.log(vec1.vec);

Constructor

Camel.Vec3(x, y, z)

The constructor of Camel.Vec3 has 3 params:

  • x:number - x-coord of the 3D vector.
  • y:number - y-coord of the 3D vector.
  • z:number - z-coord of the 3D vector.

How to use:

var vec1 = new Camel.Vec3(1.1, 2.1, 4.1);
console.log(vec1);

Methods

loadVecFloat()

Receives the Float32Array(3) value.

Return Float32Array(3) - the 3D vector value.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
console.log(vec1.loadVecFloat());

setVector(x, y, z)

Setup 3D vector value.

This method has three params:

  • x:number - x-coord of the 3D vector.
  • y:number - y-coord of the 3D vector.
  • z:number - z-coord of the 3D vector.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
vec1.setVector(1.0, 0.0, -1.0);

clone(v)

Duplicate the Vec3 object.

This method has one param:

  • v:object - if exist, this param is another Vec3 object.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
var vec2 = new Camel.Vec3(0.0, 3.0, 0.0);
vec1.clone(vec2);

add(v)

Addition the Vec3 value.

This method has one param:

  • v:object - if exist, this value has added.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
var vec2 = new Camel.Vec3(0.0, 3.0, 0.0);
vec1.add(vec2);

subtract(v)

The 3D vector subtraction.

This method has one param:

  • v:object - if exist, this value has subtracted from.

How to use:

var vec1 = new Camel.Vec3(0.0, 1.0, 0.0);
var vec2 = new Camel.Vec3(0.0, 3.0, 0.0);
vec1.subtract(vec2);
