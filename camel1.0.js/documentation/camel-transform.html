Camel.Transform

Camel.Transform is mathematic of 4x4 matrix.

Members

mx

Hold the Float32Array(16).

How to use:

var transform = new Camel.Transform();
console.log(transform.mx);

Constructor

Camel.Transform()

The constructor of Camel transform.

How to use:

var transform = new Camel.Transform();
console.log(transform);

Methods

loadMXFloat()

Receive the matrix value.

Return Float32Array(16) - A 4x4 matrix value.

How to use:

var transform = new Camel.Transform();
console.log(transform.loadMXFloat());

translate(x, y, z)

Translates the matrix.

This method has three params:

  • x:number - x-coord of the translating.
  • y:number - y-coord of the translating.
  • z:number - z-coord of the translating.

How to use:

var transform = new Camel.Transform();
console.log(transform.translate(1.0, 1.0, 1.0));
