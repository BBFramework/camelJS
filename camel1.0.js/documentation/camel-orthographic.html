Camel.Orthographic

Camel.Orthographic is the 4x4 matrix math for 3D Orthographic view.

Members

mat

Hold the Float32Array(16).

How to use:

var orthographic = new Camel.Orthographic(0.0, 0.0, 640.0, 480.0, 1.0, 1000.0);
console.log(orthographic.mat);
