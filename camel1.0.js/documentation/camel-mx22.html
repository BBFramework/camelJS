Camel.Mx22

Camel.Mx22 is the 2x2 matrix mathematics.

Members

mat

Hold the Float32Array(4).

How to use:

var vec1 = new Camel.Mx22(1.0, 2.0, 5.0, 0.0);
console.log(vec1.mat);

Constructor

Camel.Mx22(x1, y1, x2, y2)

The constructor of Camel.Mx22 has 4 params:

  • x1:number - x1-coord of the 2x2 matrix.
  • y1:number - y1-coord of the 2x2 matrix.
  • x2:number - x2-coord of the 2x2 matrix.
  • y2:number - y2-coord of the 2x2 matrix.

How to use:

var mat1 = new Camel.Mx22(1.1, 2.1, 4.1, 0.0);
console.log(mat1);

LDU(L, D, U)

Scaling the 2x2 matrix from a vector scaling This is equivalent to (but much faster than): mat2.identity(dest); mat2.scale(dest, dest, vec);

This method has three params:

  • L:Camel.Mx22 - if exist, this param is another Camel.Mx22 object.
  • D:Camel.Mx22 - if exist, this param is another Camel.Mx22 object.
  • U:Camel.Mx22 - if exist, this param is another Camel.Mx22 object.

How to use:

var mat1 = new Camel.Mx22(1.1, 2.1, 4.1, 0.0);
var l1 = new Camel.Mx22(1.1, 5.1, 2.1, 0.0);

var u1 = new Camel.Mx22(2.1, 6.1, 9.1, 1.0);
