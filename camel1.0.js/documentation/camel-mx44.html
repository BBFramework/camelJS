fromRotationTranslationScale(q, v, s)

  • v:Camel.Vec3 - if exist, this param is a 3D vector. Transition vector.
  • s:Camel.Vec3 - if exist, this param is a 3D vector. Scaling vector.

How to use:

mat1.fromRotationTranslationScale(q, vec1, vec2);

fromRotationTranslationScaleOrigin(q, v, s)

Transform a matrix from a quaternion rotation with vector translation and vector scaling.

This method has four params:

  • q:Camel.Quat - if exist, this param is a quaternion.

  • s:Camel.Vec3 - if exist, this param is a 3D vector. Scaling vector.
  • o:Camel.Vec3 - if exist, this param is a 3D vector. The origin vector around which to scale and rotate.
