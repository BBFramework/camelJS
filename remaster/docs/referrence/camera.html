Camel.Camera

Introduction

Camel.Camera is a class holds members and methods about camera in Camel.

Members

mat

Hold the Float32Array( 16 ) value.

How to use

  • var eye = new Camel.Vec3( 0.0, 2.0, 0.0 );
  • var look = new Camel.Vec3( 0.0, 2.0, 6.0 );
  • var up = new Camel.Vec3( 0.0, 1.0, 0.0 );
  • var cam = new Camel.Camera( eye, look, up );
  • console.log( cam.mat );

Constructor

Camel.Camera( eye, look, up )

Creates a new camera instance object.

The params

  • eye:Camel.Vec3 the 3d vector presents eye position.
  • look:Camel.Vec3 the 3d vector presents looking direction.
  • up:Camel.Vec3 the 3d vector presents up direction.

How to use

  • var eye = new Camel.Vec3( 0.0, 2.0, 0.0 );
  • var look = new Camel.Vec3( 0.0, 2.0, 6.0 );
  • var up = new Camel.Vec3( 0.0, 1.0, 0.0 );
  • var cam = new Camel.Camera( eye, look, up );
  • console.log( cam.mat );

The return

The new Camel.Camera instance object.
