Camel.Particle

Introduction

Camel.Particle contains the methods and the members about particle.

Extended by Camel.Transform, Camel.Geometry, and Camel.Material.

Members

visible

Holds the boolean value which specifies the particle whether is visible or is invisible. Default, it is given by true value.

How to use

  • var particle = new Camel.Particle();
  • console.log( particle.visible );

disable

Holds the boolean value which specifies the particle whether is enable or is disable. Default, it is given by false value.

How to use

  • var particle = new Camel.Particle();
  • console.log( particle.disable );

freezed

Holds the boolean value which specifies the particle whether has freezed or has not freezed. Default, it is given by false value.

How to use

  • var particle = new Camel.Particle();
  • console.log( particle.freezed );
