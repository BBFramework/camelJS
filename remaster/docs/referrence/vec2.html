Camel.Vec2

Introduction

Camel.Vec2 contains the functions about 2d mathematics.

Members

vec

It holds the Float32Array( 2 ).

How to use

  • var vec2 = new Camel.Vec2( 90.0, 10.0 );
  • console.log( vec2.vec );

Constructor

Camel.Vec2( x, y )

Create new a vector 2D instance object.

The param

  • x:number the x-coord value.
  • y:number the y-coord value.

How to use

  • var vec2 = new Camel.Vec2( 90.0, 10.0 );
  • console.log( vec2.vec );

The return

The Camel.Vec2 instance object.
