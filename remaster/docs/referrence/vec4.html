Camel.Vec4

Introduction

Camel.Vec4 contains the functions in 4d vector (quaternion) mathematics.

Members

vec

Hold the Float32Array( 4 ).

How to use

  • var vec4 = new Camel.Vec4( 1.0, 2.0, 1.5, 1.0 );
  • console.log( vec4.vec );

Constructor

Camel.Vec4( x, y, z, w )

Creates new a Camel.Vec4 instance object.

The params

  • x:number the x-coord of 4d vector.
  • y:number the y-coord of 4d vector.
  • z:number the z-coord of 4d vector.
  • w:number the w-coord of 4d vector.

How to use

  • var vec4 = new Camel.Vec4( 1.0, 2.0, 1.5, 1.0 );
  • console.log( vec4.vec );

The return

The new Camel.Vec4 instance object.
