Camel.Quat

Introduction

Camel.Quat contains functions about quaternion mathematics.

Members

vec

Hold the Float32Array( 4 ).

How to use

  • var quat = new Camel.Quat( 1.0, 2.0, 1.5, 1.0 );
  • console.log( quat.vec );

sqlerp

Performs a spherical linear interpolation with two control points..

How to use

  • var quat = new Camel.Quat( 1.0, 2.0, 1.5, 1.0 );
  • console.log( quat.sqlerp );

Constructor

Camel.Quat( i, j, k, q )

Creates new Camel.Quat instance object.

The param

  • i:number i value of the quaternion.
  • j:number j value of the quaternion.
  • k:number k value of the quaternion.
  • q:number q value of the quaternion.

How to use

  • var quat = new Camel.Quat( 1.0, 2.0, 1.5, 1.0 );
  • console.log( quat.vec );

The return

The new Camel.Quat instance object.

CalculateW()

Calculates the w component of a quat from the x, y, and z components. Assumes that quaternion is 1 unit in length. Any existing w component will be ignored.

How to use

  • var quat = new Camel.Quat( 1.0, 1.0, 1.0, 1.2 );
  • quat.CalculateW();
  • console.log( quat.vec );
