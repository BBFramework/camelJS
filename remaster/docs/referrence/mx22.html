Camel.Mx22

Introduction

Camel.Mx22 contains functions about 2x2 matrix mathematics.

Members

mat

Hold the Float32Array( 4 ).

How to use

  • var mx = new Camel.Mx22( 1.0, 2.0, 1.5, 1.0 );
  • console.log( mx.mat );

Constructor

Camel.Mx22( x1, y1, x2, y2 )

Create new the Camel.Mx22 instance object.

The params

  • x1:number the x1 component of 2x2 matrix.
  • y1:number the y1 component of 2x2 matrix.
  • x2:number the x2 component of 2x2 matrix.
  • y2:number the y2 component of 2x2 matrix.

How to use

  • var mx = new Camel.Mx22( 3.2, 1.0, 2.5, 2.3 );
  • console.log( mx.mat );

The return

The new Camel.Mx22 instance object.

LDU( L, D, U )

Scales the 2x2 matrix by the scaled 2d vector. This is equivalent to (but much faster than): Camel.Mx22.prototype.Identity, Camel.Mx22.prototype.Scale

The params

  • L:Camel.Mx22 the another 2x2 matrix for scaling.
  • D:Camel.Mx22 the another 2x2 matrix for scaling.
  • U:Camel.Mx22 the another 2x2 matrix for scaling.

How to use

  • var mx = new Camel.Mx22( 1.0, 1.2, 1.4, 0.0 );
  • var mxL = new Camel.Mx22( 1.0, 1.2, 1.4, 0.0 );
  • var mxD = new Camel.Mx22( 1.0, 1.2, 1.4, 0.0 );
  • var mxU = new Camel.Mx22( 1.0, 1.2, 1.4, 0.0 );
  • mx.LDU( mxL, mxD, mxU );

The return

The Array contains L, D, U components.
