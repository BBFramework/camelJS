Camel.Mx44

Introduction

Camel.Mx44 contains functions about 4x4 matrix mathematics.

Members

mat

Hold the Float32Array( 16 ) value.

How to use

  • var mx = new Camel.Mx44( 1.0, 1.5, 2.0, 1.0, 1.0, 2.5, 1.0, 1.5, 2.0, 2.5, 1.0, 1.5, 2.0, 1.0, 1.5, 2.0 );
  • console.log( mx.mat );

The params

  • x1:number the x1 component of the matrix value.

  • z1:number the z1 component of the matrix value.
  • w1:number the w1 component of the matrix value.
  • x2:number the x2 component of the matrix value.
  • y2:number the y2 component of the matrix value.
  • z2:number the z2 component of the matrix value.
  • w2:number the w2 component of the matrix value.
  • x3:number the x3 component of the matrix value.
  • y3:number the y3 component of the matrix value.
  • z3:number the z3 component of the matrix value.
  • w3:number the w3 component of the matrix value.
  • x4:number the x4 component of the matrix value.
  • y4:number the y4 component of the matrix value.
  • z4:number the z4 component of the matrix value.
  • w4:number the w4 component of the matrix value.

How to use

  • var mx = new Camel.Mx44( 1.0, 1.5, 2.0, 1.0, 1.0, 2.5, 1.0, 1.5, 2.0, 2.5, 1.0, 1.5, 2.0, 1.0, 1.5, 2.0 );
