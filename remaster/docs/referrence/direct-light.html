Camel.DirectLight

Introduction

Camel.DirectLight collects the methods and the members about the direct lighting.

Extended from Camel.Light.

Members

type

Holds the CAMEL_LIGHT_DIRECT const which specifies the direct light type.

How to use

  • var light = new Camel.DirectLight( 128.0, 128.0, 128.0 );
  • console.log( light.type );

You can see members of Camel.Light, Camel.Vec3, and Camel.Color for more information.

Constructor

Camel.DirectLight( x, y, z )

Creates a new Camel.DirectLight instance object.

The params

  • x:number the x-coord of the light position.
  • y:number the y-coord of the light position.
  • z:number the z-coord of the light position.

How to use

  • var light = new Camel.DirectLight( 128.0, 128.0, 128.0 );
  • console.log( light );

The return

The new Camel.DirectLight instance object.
