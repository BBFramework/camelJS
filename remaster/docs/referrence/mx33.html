Camel.Mx33

Introduction

Camel.Mx33 contains functions about 3x3 matrix mathematics.

Members

mat

Hold the Float32Array( 9 ).

How to use

  • var mx = new Camel.Mx33( 1.0, 1.5, 2.0, 1.0, 1.0, 2.5, 1.0, 1.5, 2.0 );
  • console.log( mx.mat );

Constructor

Camel.Mx33( x1, y1, z1, x2, y2, z2, x3, y3, z3 )

Create new a 3x3 matrix.

The params

  • x1:number the x1 components of 3x3 matrix.
  • y1:number the y1 components of 3x3 matrix.
  • z1:number the z1 components of 3x3 matrix.
  • x2:number the x2 components of 3x3 matrix.
  • y2:number the y2 components of 3x3 matrix.
  • z2:number the z2 components of 3x3 matrix.
  • x3:number the x3 components of 3x3 matrix.
  • y3:number the y3 components of 3x3 matrix.
  • z3:number the z3 components of 3x3 matrix.

How to use

  • var mx = new Camel.Mx33( 1.0, 1.5, 2.0, 1.0, 1.0, 2.5, 1.0, 1.5, 2.0 );
  • console.log( mx );

The return

The new Camel.Mx33 instance object.
