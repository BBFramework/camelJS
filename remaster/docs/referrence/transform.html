Camel.Transform

Introduction

Camel.Transform has 4x4 matrix mathematics about transforming.

Members

mx

Holds the Float32Array( 16 ) value.

How to use

  • var transf = new Camel.Transform();
  • console.log( transf.mx );

Constructor

Camel.Transform()

Creates a new Camel.Transform instance object.

How to use

  • var transf = new Camel.Transform();
  • console.log( transf );

The return

The new Camel.Transform instance object.

Methods

LoadMXFloat()

Receives the Float32Array( 16 ) value.

How to use

  • var transf = new Camel.Transform();
  • console.log( transf.LoadMXFloat() );

The return

The Float32Array( 16 ) value.

Identity()

Resets the Float32Array( 16 ) value.

How to use

  • var transf = new Camel.Transform();
  • transf.Identity()
  • console.log( transf.mx );

The return

The self Camel.Transform instance object.

  • y:number the y-coord for translating.
  • z:number the z-coord for translating.

How to use
