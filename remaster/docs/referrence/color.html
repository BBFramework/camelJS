Camel.Color

Introduction

Camel.Color collects the methods and the members about the color and the material.

Members

cull

Holds the boolean value which specifies assigned particle has 2-side material. Default, it is given by true value.

How to use

  • var color = new Camel.Color();
  • color.cull = false;
  • console.log( color.cull );

alpha

Holds the alpha of material. Default, it is given by 100 value.

How to use

  • var color = new Camel.Color();
  • color.SetAlpha( 100 );
  • console.log( color.alpha );

shininess

Holds the shininess of material.

How to use

  • var color = new Camel.Color();
  • color.SetShininess( 5.0 );
  • console.log( color.shininess );

Constructor

Camel.Color()

Creates a new Camel.Color instance object.

How to use

The return

The new Camel.Color instance object.

Methods

SetDiffuse( red, green, blue )

Sets diffuse color of the material.

The params

  • red:number the number presents the red channel. It limits from 0.0 value to 255.0 value.
  • green:number the number presents the green channel. It limits from 0.0 value to 255.0 value.
  • blue:number the number presents the blue channel. It limits from 0.0 value to 255.0 value.

How to use

  • var color = new Camel.Color();
  • color.SetDiffuse( 128.0, 128.0, 128.0 );

The return

The self Camel.Color instance object.
