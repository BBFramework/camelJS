Camel.AssetManager

Introduction

Camel.AssetManager makes a loading and manages the assets of a program.

Members

queueType

The array holds the asset type.

How to use

  • var assetMgr = new Camel.AssetManager( 'jpg|png|gif' );
  • console.log( assetMgr.queueType );

The number presents the counted error.

How to use

  • console.log( assetMgr.errorCount );

successCount

The number presents the counted success.

How to use

assetList

The array holds the assets has completed loading.

How to use

Constructor

Camel.AssetManager( imgExt, sndExt, xhrExt, vidExt )

Creates a new Camel.AssetManager instance object.

The params

  • imgExt:string the string presents the extensions of the image.
  • sndExt:string the string presents the extensions of the sound.
  • xhrExt:string the string presents the extensions of the xhttp.
  • vidExt:string the string presents the extensions of the video.

How to use

  • var assetMgr = new Camel.AssetManager( 'jpg|png|gif' );
  • console.log( assetMgr );

The return

The new Camel.AssetManager instance object.

QueueDownloadFile( index, type )

Downloads the asset file from the Asset Manager.

The param

  • index:number the number presents the position of the asset path.
  • type:string the string presents the asset type.

How to use

  • var assetMgr = new Camel.AssetManager( 'jpg|png|gif' );
  • assetMgr.QueueFile( 'path/asset-example.jpg' );
  • assetMgr.QueueDownloadFile( 0, CAMEL_QUEUE_IMAGE );

The return

The void return.
