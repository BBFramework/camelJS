Camel.Geometry

Introduction

Camel.Geometry contains some methods and members about the geometry.

Members

vertexBuffer

Holds the buffer which contains the vertexs.

How to use

  • var geo = new Camel.Geometry();
  • geo.CreateCell( 10.0 );
  • console.log( geo.vertexBuffer );

indiceBuffer

Holds the buffer which contains the indices.

How to use

  • var geo = new Camel.Geometry();
  • geo.CreateCell( 10.0 );
  • console.log( geo.indiceBuffer );

numberOfIndices

Holds the number of indices.

How to use

  • var geo = new Camel.Geometry();
  • geo.CreateCell( 10.0 );
  • console.log( geo.numberOfIndices );

Constructor

Camel.Geometry()

Creates new Camel.Geometry instance object.

How to use

  • var geo = new Camel.Geometry();
  • console.log( geo );

The return

The new Camel.Geometry instance object.

Methods

CreateCell( size )

Creates the cell for quad square rendering.

The param

  • size:number the number presents for size of quad square.

How to use

  • var geo = new Camel.Geometry();
  • geo.CreateCell( 10.0 );
  • console.log( geo );

The return

The self Camel.Geometry instance object.

SetModel( path )

Sets model from JSON file path to buffer.

The param

  • path:string the string presents for the path of the model file.

How to use

  • var geo = new Camel.Geometry();
  • geo.SetModel( 'http://json.com/object-model.json' );

The return

The self Camel.Geometry instance object.
