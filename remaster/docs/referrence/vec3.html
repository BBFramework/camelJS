Camel.Vec3

Introduction

Camel.Vec3 contains the functions in 3d vector mathematics.

Members

vec

Hold the Float32Array( 3 ).

How to use

  • var vec3 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • console.log( vec3.vec );

Constructor

Camel.Vec3( x, y, z )

Creates new a Camel.Vec3 instance object.

The params

  • x:number the x-coord of 3d vector.
  • y:number the y-coord of 3d vector.
  • z:number the z-coord of 3d vector.

How to use

  • var vec3 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • console.log( vec3.vec );

The return

The new Camel.Vec3 instance object.

Hermite( a, b, c, d, t, out )

Performs a linear interpolation between two control point.

The params

  • a:Camel.Vec3 the first operand.
  • b:Camel.Vec3 the second operand.
  • c:Camel.Vec3 the third operand.
  • d:Camel.Vec3 the fourth operand.
  • t:number the interpolation time.
  • out:Camel.Vec3 if it has been set, if true value returns the new result, otherwise it changes value of current object or changes value of out object.

How to use

  • var vec1 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • var vec2 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • var vec3 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • var vec4 = new Camel.Vec3( 1.0, 2.0, 1.5 );
  • var vec5 = new Camel.Vec3( 1.2, 2.5, 3.5 );
  • var hermiteVec = vec1.Hermite( vec2, vec3, vec4, vec5, dt );
  • console.log( hermiteVec );

The return

The new Camel.Vec3 instance object.
