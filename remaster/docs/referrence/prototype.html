Camel Prototype

Introduction

Camel Prototype is a collection of members and methods in the Camel class.

Members

id

Hold the canvas id.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • console.log( engine.id );

clearColor

The array holds color which camel use to clear the context. This array has three elements presents for three channels such as R, G, B.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • engine.SetClearColor( 255, 0, 0 );
  • console.log( engine.clearColor );

renderHolder

The array holds renderers camel use to render out from the context. Defaultly, this array has eight elements.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • console.log( engine.renderHolder );

Constructor

Camel( CANVASElementID, settings, extensions, numberHolder )

The construct of Camel has 4 params.

The params

  • CANVASElementID:string the id string of canvas developer want to use with Camel.
  • settings:object the settings object developer want to config for Camel.
  • extensions:array an array collects WebGL extensions.
  • numberHolder:number number of render. Maximum is 4 value

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • console.log( engine );

The return

The Camel instance object.

BuildScene( renderer, startCB, updateCB, renderCB, beforeRenderCB )

Builds the scene.

The params

  • renderer:object the renderer used to render the scene.
  • startCB:callback the starting steps of the scene.
  • updateCB:callback the update callback when the scene has being rendered.
  • renderCB:callback the rendering steps.
  • beforeRenderCB:callback the callback function executed before rendering.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • var scene = engine.BuildScene(
  •     CAMEL_RENDERER_TEXTURE,
  •     function()
  •     {
  •         // Some start code of the scene...
  •     }
  • );

The return

The Camel.Scene instance object.

OrderRender( renderer )

Registry the renderer into Camel render machine.

The param

  • renderer:object the needle renderer.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • console.log( engine.OrderRender( CAMEL_RENDERER_TEXTURE ) );

The return

The renderer object. In fail case, it returns false value.

BuildRender( shaders, constants, beforeCB, finishCB )

Builds the renderer.

The params

  • shaders:object the shaders used to render the scene.
  • constants:object the object contains the members are constants buffer of shader.
  • beforeCB:callback the action has be executed when the scene has being rendered.
  • finishCB:callback the action has be executed when the scene renders completely.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • var scene = engine.BuildRender(
  •     [ vs, fs ]
  •     {
  •         'positionIn':CAMEL_ATTRIB
  •         'texCoordIn':CAMEL_ATTRIB
  •     }
  • );

The return

The renderer object. In fail case, it returns false value.

CreateRTT( name, width, height, red, green, blue, alpha, render )

Creates dynamic textures.

The params

  • name:string the string presents name of the texture.
  • width:number the number presents the length of width of the texture.
  • height:number the number presents the length of height of the texture.
  • red:number the number presents the red channel of swap color.
  • green:number the number presents the green channel of swap color.
  • blue:number the number presents the blue channel of swap color.
  • alpha:number the number presents the alpha channel of swap color.
  • render:object the renderer used to render the scene.

How to use

  • var engine = new Camel( 'canvas_id', { antialias: false, [], 4 } );
  • engine.CreateRTT( 'texName', 255, 255, 255, 0, 0, 100, renderer );

The return

The WebGLFrameBuffer instance object.
