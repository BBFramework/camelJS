Camel.Orthographics

Introduction

Camel.Orthographics is a class contains members and mothods about orthographics. It includes many 4x4 matrix mathematics also.

Members

mat

Hold the Float32Array( 16 ) value.

How to use

  • var mx = new Camel.Orthographics( 0.0, 0.0, 640.0, 480.0, 1.0, 1000.0 );
  • console.log( mx.mat );

  • right:number the number presents the right of view.
