Camel.Mx23

Introduction

Camel.Mx23 contains functions about 2x3 matrix mathematics.

Members

mat

Hold the Float32Array( 6 ).

How to use

  • var mx = new Camel.Mx23( 1.0, 2.0, 1.5, 1.0, 1.5, 1.0 );
  • console.log( mx.mat );

Constructor

Camel.Mx23( x1, y1, z1, x2, y2, z2 )

Create new a Camel.Mx23 instance object.

The params

  • x1:number the x1 component of 2x3 matrix.
  • y1:number the y1 component of 2x3 matrix.
  • z1:number the z1 component of 2x3 matrix.
  • x2:number the x2 component of 2x3 matrix.
  • y2:number the y2 component of 2x3 matrix.
  • z2:number the z2 component of 2x3 matrix.

How to use

  • var mx = new Camel.Mx23( 1.0, 2.0, 1.5, 1.0, 1.5, 1.0 );
  • console.log( mx );

The return

The new Camel.Mx23 instance object.

FromRotation( deg )

Rotates the 2x3 matrix by the given angle. This is equivalent to (but much faster than): Camel.Mx23.prototype.Identity, Camel.Mx23.prototype.Rotate.

The param

  • deg:number the rotation angle.

How to use

  • var mx = new Camel.Mx23( 1.0, 2.0, 1.5, 1.0, 1.5, 1.0 );
  • var vec = new Camel.Vec2( 1.0, 2.0 );
  • mx.FromRotation( vec );

The return

The self Camel.Mx23 instance object.
