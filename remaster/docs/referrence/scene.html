Camel.Scene

Introduction

Camel.Scene has used to build a scene. It includes the follow methods and members

Members

visible

Holds the flag which specifies whether the scene is visible or is invisible. Default, it had set to true value.

How to use

  • var scene = new Camel.Scene();
  • console.log( scene.visible );

disable

Holds the flag which specifies whether the scene is disable or is enable. Default, it had set to false value.

How to use

  • var scene = new Camel.Scene();
  • console.log( scene.disable );

particleHolder

The array holds the particle items. Default, it has the length to 32 value.

How to use

  • var scene = new Camel.Scene();
  • console.log( scene.particleHolder );

Constructor

Camel.Scene( start, update, render, beforeRender )

Creates new Camel.Scene instance object.

The params

  • start:function the function initializes the scene.
  • update:function the function updates the scene.
  • render:function the function renders the scene.
  • beforeRender:function the function has executed before each render time.

How to use

  • var scene = new Camel.Scene();
  • console.log( scene );

The return

The new Camel.Scene instance object.

Methods

Enable()

Enables the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.Enable();

The return

The void return.

Disable()

Disables the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.Disable();

The return

The void return.

AddRenderer( renderer )

Add a renderer to list of renderers.

The param

  • renderer:object the renderer used to render the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.AddRenderer( CAMEL_RENDERER_TEXTURE );

The return

The self Camel.Scene instance object.

Pass( renderer )

Render the scene.

The param

  • renderer:object the renderer used to render the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.Pass( CAMEL_RENDERER_TEXTURE );

The return

The void return.

AddChild( particle )

Adds a particle to the scene.

The param

  • particle:object the particle has been render by the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.AddChild( particle );

The return

The object had added.

AddLight( light )

Adds a light to the scene.

The param

  • light:object the light has been render by the scene.

How to use

  • var scene = new Camel.Scene();
  • scene.AddLight( light );

The return

The light had added.
