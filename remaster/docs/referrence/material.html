Camel.Material

Introduction

Camel.Material contains a lot of methods and members about material.

Extended by Camel.Color

Members

map

The object holds the texture mapping.

How to use

  • var mat = new Camel.Material();
  • console.log( mat.map );

You can see members of Camel.Color for more information.

Constructor

Camel.Material()

Creates a new Camel.Material instance object.

How to use

  • var mat = new Camel.Material();
  • console.log( mat );

The return

The new Camel.Material instance object.

Methods

SetDiffuseMap( map )

Sets diffuse texture mapping to the material.

The param

  • map:object the texture mapping.

How to use

  • var mat = new Camel.Material();
  • mat.SetDiffuseMap( engine.GetTexture( assetMgr.GetAsset( 'images/fish-tex.png' ) );
  • console.log( mat.map.diffuse );

The return

The self Camel.Material instance object.
