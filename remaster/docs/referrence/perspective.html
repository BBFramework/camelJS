Camel.Perspective

Introduction

Camel.Perspective is a class contains members and methods about 3d perspective. It includes many 4x4 matrix mathematics also.

Members

mat

Hold the Float32Array( 16 ) value.

How to use

  • var mx = new Camel.Perspective( 45.0, 1.0, 1.0, 100.0 );
  • console.log( mx.mat );

Constructor

Camel.Perspective( fov, ratio, near, far )

Creates a new 3d perspective instance object.

The params

  • fov:number the angle of field of view.
  • ratio:number the ratio between width and height of perspective.
  • near:number the nearest view of perspective.
  • far:number the farest view of perspective.

How to use

  • var p = new Camel.Perspective( 45.0, 1.0, 1.0, 100.0 );
  • console.log( p );

The return

The new Camel.Perspective instance object.
