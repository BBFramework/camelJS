Camel.Light

Introduction

Camel.Light collects the methods and the members about the basic lighting.

Extended from Camel.Vec3, Camel.Color.

Members

You can see members of Camel.Vec3, Camel.Color for more information.

Constructor

Camel.Light( x, y, z )

Creates a new Camel.Light instance object.

The params

  • x:number the x-coord of the light position.
  • y:number the y-coord of the light position.
  • z:number the z-coord of the light position.

How to use

  • var light = new Camel.Light( 128.0, 128.0, 128.0 );
  • console.log( light );

The return

The new Camel.Light instance object.
